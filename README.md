Description: Recipe Finder is a React web application that helps users discover recipes based on ingredients they have at home. Users can enter an ingredient (chicken, for example) and the application will show an assortment of different recipes that use that ingredient. They can click on a specific recipe to view other ingredients needed as well as the steps needed to prepare the dish. This simple layout and functionality makes the application easy to use for busy college students and beginner cooks. 

How to Run: 
1. download or clone the projects files and repository
2. open the project folder in VSC
3. open a terminal in VSC
4. start the development server by running "npm run dev"
5. open the local URL displayed in the terminal

Sources: This web application uses data provided by TheMealDB public API. https://www.themealdb.com/
Recipe names, images, ingredients, and cooking instructions are retrieved dynamically from TheMealDB using JavaScript's fetch() function.

AI was used to assist me with the funcionality of the application, to troubleshoot errors, and with using fetch(). I typed everything out by hand and tested everything myself.
